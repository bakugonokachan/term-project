## Install

1.1 安裝必要套件

若未安裝 Matplotlib，請執行：

pip install matplotlib

1.2 執行程式
python3 B11504118_term\ project.py

1.3 互動操作

執行後會開啟視窗，內含三組滑桿：

| 滑桿         | 說明              |
| ----------- | --------------- |
| v0 (m/s)    | 調整初速（1–100 m/s） |
| Angle (deg) | 調整發射角度（1–89°）   |
| Drag k      | 調整線性空氣阻力（0–1）   |
